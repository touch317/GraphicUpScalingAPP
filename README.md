# 🚀 簡単画像アップスケーリング

LANCZOS法とシャープネス処理を組み合わせた、高品質な画像拡大Webアプリです。
ブラウザ上で複数の画像をまとめて処理し、ビフォーアフターを比較しながらダウンロードできます。

## ✨ 主な機能
- **自由な拡大倍率**: 1.0倍から8.0倍まで、0.5刻みで調整可能。
- **鮮明化調整**: 拡大時のぼやけを解消するシャープネス機能。
- **一括変換**: 複数の画像をドラッグ＆ドロップでまとめてアップスケーリング。
- **比較スライダー**: 元画像と加工後を左右にスライドして比較。

## 🛠 使用技術
- Python
- Streamlit (Web Framework)
- Pillow (Image Processing)
- Streamlit Image Comparison (UI Component)
